# AOSP ROM Paylaşım Sitesi

Bu proje, farklı AOSP ROM topluluklarının `device.json` kaynaklarını çekip tek ekranda gösteren statik bir web sayfasıdır.

## Desteklenen ROM kaynakları

- LineageOS
- AlphaDroid
- AxionOS
- Project Infinity
- YAAP

## Çalıştırma

Bu bir statik sitedir. Klasörü bir web sunucusuyla açmanız yeterli.

Örnek:

```bash
python3 -m http.server 4173
```

Sonrasında `http://localhost:4173` adresini açın.
